# Used-Car-Overprice-Detector-for-the-Indian-Market
 Built a Random Forest regression model using scikit-learn to predict used car prices on 6000+ listings  
• Achieved R² ≥ 0.70, explaining close to 70% of price variation in car prices and indicating strong model performance 
• Developed end-to-end data pipeline using pandas and numpy for feature engineering (brand extraction, unit parsing), handling 
missing values, and LabelEncoder for categorical variables 
• Created a Flask REST API with /predict endpoint that classifies vehicles as overpriced, underpriced, or fairly priced based on 15% deviation threshold from predicted value; used joblib for model serialization 
• Designed a responsive web interface with a modern UI, real time form validation and animated result displays using 
HTML/CSS/JavaScript 
• Performed exploratory data analysis with pandas, matplotlib and seaborn to identify pricing patterns across brands, fuel types and transmission categories
